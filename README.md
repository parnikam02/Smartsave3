# 💰 SmartSave – Intelligent Personal Finance Manager

SmartSave is an AI-powered personal finance management application designed to help users track, analyze, and optimize their financial activities in one place. It provides smart insights, budgeting tools, and automated savings to promote better financial habits.

---

## 🚀 Features

### 🔗 Multi-Platform Integration
- Connect with digital payment apps (GPay, PhonePe)
- View all bank transactions in one unified dashboard  
- No sensitive personal data is exposed

### 📊 Smart Expense Tracking
- Automatically categorizes transactions (Food, Travel, Bills, etc.)
- Tracks daily, weekly, and monthly spending
- Manual entry for cash transactions

### 📅 Budget Planning
- Monthly planner based on user income
- Suggests ideal spending percentages
- Helps manage expenses efficiently

### 🤖 AI Financial Assistant
- Provides personalized financial insights
- Suggests better spending and saving habits

### 💸 Micro-Savings System
- Save small amounts daily (e.g., ₹10/day)
- Choose which bank account to save from
- Encourages consistent saving habits

### 🎯 Goals & Rewards
- Set monthly budget goals
- Earn rewards on achieving targets

### 🧾 Bill Payments & Recharge
- Pay utility bills and recharge directly
- Track all payments in one place

### 🧍 Borrow & Lend Tracker
- Record borrowed/lent money
- Get reminders for repayments

### 🧑‍💻 Admin Panel
- View user activity (usage only)
- Ensures privacy-focused monitoring

---

## 🧠 How It Works

1. User logs in and connects payment platforms  
2. Transactions are fetched and categorized automatically  
3. AI analyzes spending patterns  
4. User sets budget and savings goals  
5. SmartSave provides insights, reminders, and financial plans  

---

## 🛠️ Tech Stack

- **Frontend:** React (TypeScript)
- **Routing:** React Router
- **Styling:** Tailwind CSS
- **Charts & Analytics:** Recharts
- **AI Integration:** Google GenAI API
- **State Management:** Local Storage (custom services)
